The Read Finder

In the Projects tab, select the File menu:

When you select the Show read finder option, Minerva displays a new tab:

The read finder enables you to search Arcturus for one or more reads, to discover which contigs they have been assembled into.

You can type or paste a list of read names into the Reads to search for box. You can also import a list of read names from a file using the Open file of read names menu option:

The Find reads button becomes active when one or more read names are entered into the Reads to search for box. When the button is pressed, Minerva searches for the reads and displays the results in the Information box:
