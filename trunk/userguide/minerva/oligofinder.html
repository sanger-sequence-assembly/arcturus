The Oligo Finder

In the Projects tab, select the Project menu:

When you select the Find oligos option, Minerva displays a new tab:

The oligo finder enables you to search Arcturus for one or more oligo sequences.

Minerva can search through the contig consensus sequences in one or more projects. It can also search through the unassembled reads.

This can be a lengthy operation, so Minerva uses progress bars to allow you to monitor the completion of the search.

Specifying the oligos

Oligos are entered, one per line, into the Oligos to search for box. You can enter the oligo sequence alone, or a name followed by a sequence. Minerva will automatically assign a name to each anonymous oligo.

Oligo sequences should contain only the four bases A, C, G and T. Minerva does not allow you to include pads in oligo sequences.

You can also import a set of oligos from a file using the Open file of oligos menu option:

Choosing which contigs to search

Minerva displays a list of projects to the right of the Oligos to search for box. It will search the contigs in all of the projects that you select.

For convenience, you can use the All projects check-box to select all of the projects. In this mode, Minerva will search all of the current contigs.

Searching the unassembled reads

You may also wish to search the unassembled reads. Use the Scan free reads check-box to tell Minerva to do this.
