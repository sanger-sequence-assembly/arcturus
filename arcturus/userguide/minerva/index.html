Introduction to Minerva

Welcome to Minerva, the graphical user interface to Arcturus.

The web pages which follow explain how to use Minerva effectively to manage your assembly projects.

Using Minerva, you will be able to:

  • View the current state of one or more projects in any assembly.
  • Examine the contigs in any projec.
  • Search for oligo sequences in the current contigs and/or the unassembled reads.
  • Query the status of any read by name.
  • Request the transfer of one or more unassembled reads into a particular project.
  • Request the transfer of contigs between projects.

What is a project?

In Arcturus, a project is a group of contigs which are being worked on by a finisher.

In most cases, it's the same thing as a "split" database, although Arcturus allows you to create new projects to enable you to group your contigs in different ways, so you could have a "split" which contains several Arcturus projects, and Minerva will display each project separately.
