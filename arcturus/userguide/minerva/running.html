Running Minerva

You can run Minerva on your desktop machine. Open a terminal window on your desktop and type the command

/software/arcturus/java/scripts/minerva

Alternatively, copy this file to your Desktop directory. It will place an icon on your desktop, and you can run Minerva by simply double-clicking the icon.

Whichever way you start Minerva, you will see the Minerva logo appear for a few seconds whilst Minerva starts up:
