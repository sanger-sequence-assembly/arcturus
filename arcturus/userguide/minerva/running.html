Running Minerva

You can run Minerva on your desktop machine. Open a terminal window on your desktop and type the command

java -jar /software/arcturus/java/arcturus.jar
You will see the Minerva logo appear for a few seconds whilst Minerva starts up:

Then you will see this window, which shows all of the organisms which are known to Arcturus:
