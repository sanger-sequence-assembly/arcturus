The Read Importer

In the Projects tab, choose a project and then select the Project menu:

When you select the Import reads into project option, Minerva displays a new tab:

The read importer enables you to assign one or more unassembled reads to a project. Each read is converted into a single-read contig, and the contig is added to the project.

Specifying the reads

Read names are entered, one per line, into the Reads to import box. Wildcards are not permitted.

Wildcards are not permitted. Each read name must identify a single read.

You can also import a list of read names from a file using the Open file of read names menu option:

Specifying the project

Minerva displays a list of projects to the right of the Reads to import box. Use this list to select the project where you want Minerva to put the reads.

Importing the reads

When you press the Import reads button, Minerva will attempt to import each read in the list into the project you have specified.

An import will fail if the read has not been loaded into Arcturus, or if the read is already in a current contig. This includes single-read contigs, so you cannot use the read importer to move single-read contigs from one project to another.

Each read is processed separately. If one read cannot be imported, Minerva will report the reason, and proceed to try to import the next read in the list.
